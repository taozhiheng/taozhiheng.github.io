###adb常用shell命令

#adb shell …

  1. cd path

  2. ls

  3. mkdir dir

  4. touch file

  5. echo “….” > file　写入

  6. cat file　查看

  7. cp file file_copy

  8. mv filePath targetPath

  9. rm file

  10. more file　分页显示

  11. wc file　显示行数、字数、字符数

  12. find -name file

  13. grep string file　查找字符串

  14. pwd　显示目录

  15. tree　显示目录树

  16. ln -s path1 path2 建立软连接

  17. stat filePath　显示文件信息

  18. whoami　显示用户身份

  19. hostname　显示主机名

  20. hostname -i 显示主机ip

  21. uname 显示系统信息

  22. uname -a 显示系统全部信息

  23. top　动态显示耗费资源最多进程，实时监控

  24. ps　显示瞬间进程状态

  25. -aux　显示锁哦有书剑进程状态

  26. du file(dir)　显示文件（目录）已用磁盘空间总量

  27. df　显示文件系统磁盘空间使用情况

  28. free　显示内存和交换空间使用情况

  29. ifconfig　显示网络接口信息

  30. ping ip　测试网络连通性

  31. netstat　显示网络状态
